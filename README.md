# my-chess-game
# ♟️ Grandmaster Chess Pro

A high-performance, serverless chess application built with **C++ (Stockfish WASM)** and **JavaScript**. Play against a world-class engine, challenge friends via P2P, or analyze your games—all directly in your browser.

[**🚀 PLAY NOW**](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)

---

## 🌟 Key Features

*   **🤖 Advanced Bot:** Powered by **Stockfish 10 (WebAssembly)**. Choose your challenge level from 400 Elo (Beginner) up to 2400 Elo (Grandmaster).
*   **🤝 P2P Multiplayer:** Challenge friends anywhere in the world using direct Peer-to-Peer connections (no account required).
*   **🔎 Game Review:** Post-game analysis that detects blunders and highlights "Best" moves, just like professional platforms.
*   **📏 FIDE Compliant:** Full support for castling, en passant, stalemate, and the 50-move rule.
*   **📱 Mobile Friendly:** Fully responsive UI designed to work on phones, tablets, and desktops.

## 🛠️ Built With

*   **Logic:** [chess.js](https://github.com/jhlywa/chess.js) (FIDE Rules Engine)
*   **Engine:** [Stockfish.js](https://github.com/nmrugg/stockfish.js) (Compiled to WASM)
*   **Networking:** [PeerJS](https://peerjs.com/) (WebRTC for P2P)
*   **UI:** [Chessboard.js](https://chessboardjs.com/)
